# Project template scaffold

This is the baseline scaffold for new projects used by `project-setup` and `dev` skills.

## Included
- `.ai/` control-plane directories
- starter gate configs (`.ai/pre-commit.json`, `.ai/pre-push.json`)
- starter rules/spec/log docs

## First-run checklist
1. Replace placeholder commands in `.ai/pre-commit.json`.
2. Add project spec docs under `.ai/spec/`.
3. Install/verify local git hooks.
4. Confirm readiness artifact generation under `.ai/log/`.
