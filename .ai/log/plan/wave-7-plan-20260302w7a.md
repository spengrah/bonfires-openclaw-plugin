# Wave 7 Plan (20260302w7a)

## Objective
Harden bonfires-plugin for first-class OpenClaw plugin distribution and operator clarity.

## Scope (in)
1. Add OpenClaw plugin metadata for install/discovery (`openclaw.plugin.json`, package metadata).
2. Add/validate env-friendly fallback behavior for base URL and credential-selector fields (`baseUrl`, `apiKeyEnv`, `bonfireId`) without requiring plaintext config values.
3. Document heartbeat naming collision and subsystem boundaries (Gateway heartbeat vs plugin stack heartbeat).
4. Define and implement plugin state persistence policy aligned with runtime operation (plugin-scoped durable paths, configurable overrides).
5. Align background loops with plugin lifecycle services where OpenClaw plugin API supports deterministic start/stop semantics.
6. Update README/operator docs with install, config, and troubleshooting paths.

## Non-goals (out)
1. No new Bonfires product capabilities beyond packaging/integration hardening.
2. No rework of retrieval/capture/recovery semantics already shipped.
3. No redesign of OpenClaw core heartbeat subsystem.

## Pass/fail gates
1. `npm run lint` => PASS.
2. `npm run test` => PASS.
3. `npm run verify:hosted` => PASS (fixture mode).
4. `npm run gate:all` => PASS.
5. Plugin metadata validates under OpenClaw plugin doctor/install flow (`openclaw plugins doctor`) in local packaging test.

## Open blockers
1. Final runtime path convention choice for plugin-owned state dir (depends on exact runtime helper availability and plugin deployment layout).
2. Clarifying whether lifecycle services should fully own heartbeat/ingestion loops or if wrapper registration is needed for existing startup model.

## Assumption checks
1. **Baseline green:** PASS (`npm run gate:all`).
2. **Scope = risk scope:** yes; matrix covers packaging, config/env ergonomics, lifecycle, and operator docs.
3. **Gate coverage:** yes; all acceptance criteria map to runnable checks or plugin doctor validation.
4. **Spec/guidance alignment:** yes; PM5 spec + PM5 guidance added as paired docs.
